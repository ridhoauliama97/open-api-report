- Base host: `http://192.168.10.100:5006`
- Prefix API: `/api`
- Pola akses PDF: `GET|POST /api/reports/pps/{path}/pdf`

- Laporan PPS Bahan Baku Mutasi Bahan Baku: `http://192.168.10.100:5006/api/reports/pps/bahan-baku/mutasi-bahan-baku/pdf`

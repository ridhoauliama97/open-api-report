# POST `/api/reports/sawn-timber/st-sawmill-hari-tebal-lebar`

Preview laporan ST Sawmill / Hari / Tebal / Lebar.

## Parameters

- `TglAwal` (date, required)
- `TglAkhir` (date, required)

## Response

Mengembalikan JSON berisi data mentah dari SP, daftar tanggal, data pivot per group/tebal/lebar, dan ringkasan.

# POST `/api/reports/sawn-timber/st-sawmill-hari-tebal-lebar/health`

Health check struktur output SP untuk laporan ST Sawmill / Hari / Tebal / Lebar.

## Parameters

- `TglAwal` (date, required)
- `TglAkhir` (date, required)

## Response

Mengembalikan status apakah struktur kolom output masih sesuai konfigurasi.
